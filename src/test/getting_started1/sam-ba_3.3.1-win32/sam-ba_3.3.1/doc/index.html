    Microchip SAM Boot Assistant (SAM-BA) software provides an open set of tools for programming Microchip ARM Thumb-based microprocessors. Once the SAM-BA monitor, a in-system programmer, is executed on the target device, customers can use SAM-BA as a tool to re-program the MPU, without connecting to the debugger.
