    SAM-BA readuniqueid applet

    Reading the device unique ID

    The readuniqueid applet displays the device unique ID and optionally saves it into a binary file.

    sam-ba -p <port> -d <device> -a readuniqueid[:<filename>]

$ sam-ba -p serial -d sama5d2 -a readuniqueid
$ sam-ba -p serial -d sama5d2 -a readuniqueid:uid.bin
